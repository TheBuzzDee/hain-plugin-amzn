# hain-plugin-amzn
[Hain](https://github.com/appetizermonster/hain) plugin to search amazon.com

![screenshot](screenshot.png)

## Install

Type this in the Hain input:
```
/hpm install amzn
```

## Usage

Type /amzn yoursearch:
```
/amzn yoursearch
```
